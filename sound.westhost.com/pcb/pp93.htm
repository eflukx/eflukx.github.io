Elliott Sound Products Project 93 - Microphone Amplifier 

This tiny board (13 x 50mm) is a complete balanced microphone capsule amplifier, with phantom feed powering. It is suitable for use with measurement or recording microphones, and has high output and excellent linearity. It is intended for use with electret microphone capsules.

The layout comprises one microphone amplifier. The latest PCB is double-sided with plated-through holes and is solder masked on both sides.
