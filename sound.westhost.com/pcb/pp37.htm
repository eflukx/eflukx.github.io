Elliott Sound Products Project 37A - Minimalist (DoZ) Hi-Fi Preamp

New Version Now Available

The circuit for this amp is identical to the published (Revision A) project. The performance is really quite exceptional for such a simple circuit, and I have had many e-mails praising the sound of the original version of this preamp. The new version has virtually identical performance, but now uses a dual supply (±15V recommended).

The layout comprises two channels. Multiple units can be powered from the one P05 (Rev-A) power supply which is available separately.
