Elliott Sound Products Project 19 - Single Chip 50W Amplifier

Revision B boards are now being supplied. See project details for information.

The layout for this amp is changed from the originally published project. The SIM connections have been deleted, and you may now use PCB headers for input an power, and a quick-connect header for the speaker output.

The layout comprises two amplifiers, individually fused on board.  The board is designed to be able to be cut down the middle for separate amplifiers.

Although this is a very simple amplifier to build, sound quality is very good, and it is an ideal project for those who want to start building their own gear, but still want a quality amplifier. LM3886 power opamps can also be used with the addition of a wire link on the PCB.
