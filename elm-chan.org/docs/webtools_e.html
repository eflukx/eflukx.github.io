Page Creation Tools

These are the software tools that I am using to create the pages in this site.

Wz Editor (Village Center)

Generic text editor using to write html code directry. This enables to use full features of html and css.

BSCH (Suigyodo)

A drawing program optimized for schematic. This is one of the useful schematic CADs for web creation because it can output the schematic in bitmap format.

Paint (Windows)

A simple drawing tool included in Windows. It has sufficient functions to draw simple diagrams.

Painter IX (Corel)

An excellent natural media painting tool. Recently I transferred version from 5 to 9. Of course I have a PTZ-630 graphics tablet, too.

D-ORDERS! (LNsoft Entertainment)

Image format converter. It can be used in one stroke operation, D&D or Send To. A free version is also available.

FFFTP (sota)

An ordinary FTP client program.
